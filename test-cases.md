# 测试用例

## 1. 文件夹扫描

| 编号 | 用例 | 预期结果 |
|------|------|----------|
| 1.1 | 点击"打开文件夹"，选择含 mp3/flac/wav/ogg 文件的目录 | 列表显示所有支持格式的文件 |
| 1.2 | 选择空文件夹 | 列表为空，无报错 |
| 1.3 | 选择含子目录的文件夹 | 仅扫描顶层文件（不递归） |
| 1.4 | 取消文件夹选择对话框 | 无变化，无报错 |

## 2. 音频播放

| 编号 | 用例 | 预期结果 |
|------|------|----------|
| 2.1 | 点击列表中的曲目 | 开始播放，底部控制栏显示文件名 |
| 2.2 | 点击播放/暂停按钮 | 切换播放/暂停状态 |
| 2.3 | 点击上一首/下一首 | 切换到对应曲目并播放 |
| 2.4 | 拖动进度条 | 跳转到对应位置继续播放 |
| 2.5 | 拖动音量滑块 | 音量实时变化 |
| 2.6 | 播放结束 | 自动切换到下一首 |
| 2.7 | 最后一首播放结束 | 停止播放（不循环） |

## 3. 歌词同步

| 编号 | 用例 | 预期结果 |
|------|------|----------|
| 3.1 | 播放有对应 LRC 文件的曲目 | 主窗口显示歌词，当前行高亮并自动滚动 |
| 3.2 | 播放无 LRC 文件的曲目 | 显示"暂无歌词" |
| 3.3 | 列表中有 LRC 的曲目 | 显示"词"标识 |
| 3.4 | 拖动进度条 | 歌词跳转到对应位置 |

## 4. 桌面歌词

| 编号 | 用例 | 预期结果 |
|------|------|----------|
| 4.1 | 点击桌面歌词开关 | 悬浮窗口出现，显示当前歌词 |
| 4.2 | 再次点击桌面歌词开关 | 悬浮窗口隐藏 |
| 4.3 | 拖拽桌面歌词窗口 | 窗口跟随移动 |
| 4.4 | 桌面歌词窗口显示时切歌 | 歌词同步更新 |
| 4.5 | 桌面歌词置顶 | 始终在其他窗口之上 |

## 5. 系统托盘

| 编号 | 用例 | 预期结果 |
|------|------|----------|
| 5.1 | 点击窗口关闭按钮 | 窗口隐藏到托盘，不退出 |
| 5.2 | 左键点击托盘图标 | 显示主窗口 |
| 5.3 | 右键托盘 → 播放/暂停 | 切换播放状态 |
| 5.4 | 右键托盘 → 显示主窗口 | 窗口恢复显示并聚焦 |
| 5.5 | 右键托盘 → 桌面歌词 | 切换桌面歌词窗口显示/隐藏 |
| 5.6 | 右键托盘 → 退出 | 应用完全退出 |

## 6. 自定义标题栏

| 编号 | 用例 | 预期结果 |
|------|------|----------|
| 6.1 | 拖拽标题栏 | 窗口跟随移动 |
| 6.2 | 点击最小化 | 窗口最小化到任务栏 |
| 6.3 | 点击关闭 | 窗口隐藏到托盘 |
